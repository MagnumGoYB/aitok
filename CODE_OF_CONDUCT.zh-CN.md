# Code of Conduct

[English](CODE_OF_CONDUCT.md)

## 行为标准

本项目欢迎尊重、建设性的协作。贡献者应聚焦问题本身，清楚说明取舍，避免骚扰、侮辱、歧视或披露私人信息。

## 期望行为

- 在 issue、review 和讨论中保持尊重和具体。
- 反馈应可执行，并围绕项目目标。
- 保护用户隐私，从示例中移除敏感数据。
- 尊重维护者的范围判断和 review 时间。

## 不可接受行为

- 骚扰、威胁、侮辱或歧视性语言。
- 发布私人信息、prompt、日志、API Key 或凭据。
- 反复发布偏题内容，或施压绕过安全和验证规则。
- 滥用自动化、刷屏或故意破坏项目秩序。

## 执行

维护者可以编辑或删除评论、关闭 issue、阻止用户，或采取其他合理措施保护项目。可通过仓库维护者或 GitHub moderation 工具报告问题。
