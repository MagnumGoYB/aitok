# Code of Conduct

[中文](CODE_OF_CONDUCT.zh-CN.md)

## Our Standard

This project welcomes respectful, constructive collaboration. Contributors should focus on the work, explain tradeoffs clearly, and avoid harassment, insults, discrimination, or disclosure of private information.

## Expected Behavior

- Be respectful and specific in issues, reviews, and discussions.
- Keep feedback actionable and grounded in project goals.
- Protect user privacy and remove sensitive data from examples.
- Respect maintainers' scope decisions and review bandwidth.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing private information, prompts, logs, API keys, or credentials.
- Repeated off-topic comments or pressure to bypass security and validation rules.
- Abusive automation, spam, or intentionally disruptive behavior.

## Enforcement

Maintainers may edit or delete comments, close issues, block users, or take other reasonable action to protect the project. Report concerns through the repository maintainers or GitHub moderation tools.
