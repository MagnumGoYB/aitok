## Getting Help

Use GitHub issues for bug reports, feature requests, and documentation gaps. Please use the issue templates when available.

## Before Opening an Issue

- Run `aitok doctor` when the problem involves local data discovery.
- Run `make test` when reporting a development regression.
- Remove API keys, prompts, private paths, and personal data from logs.
- Include the command, period, tool, expected result, actual result, and operating system.

## Scope

Maintainers can help with `aitok` parsing, CLI output, local setup, and release artifacts. They cannot recover historical Gemini CLI usage that was not recorded by local telemetry.
